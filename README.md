# 🎓 IFRI MentorLink

**IFRI MentorLink** est une application web conçue pour mettre en relation les étudiants de l'Institut de Formation et de Recherche en Informatique (IFRI) souhaitant bénéficier ou offrir du mentorat académique et professionnel.

Ce projet a été réalisé dans le cadre de l'Unité d'Enseignement "Projet Intégrateur" de l'année académique 2025-2026.

## ✨ Fonctionnalités Principales

- **Gestion des Profils** : Création de profils personnalisés avec définition du rôle (Mentor/Mentoré), de la filière, du niveau d'études, ainsi que des compétences et lacunes.
- **Offres et Demandes** : Publication et consultation d'annonces de mentorat ciblées par matière, format (en ligne/présentiel) et disponibilités horaires.
- **Système de Matching** : Algorithme de mise en relation permettant de postuler aux annonces et d'accepter/refuser des candidatures de mentorat.
- **Messagerie Instantanée** : Chat intégré en temps réel pour organiser et suivre les sessions de mentorat une fois le match validé.

## 🛠️ Technologies Utilisées

- **Frontend** : HTML5, CSS3, JavaScript (Architecture modulaire et responsive).
- **Backend** : Python 3 avec le framework Django.
- **Base de données** : MySQL (SGBD Relationnel).
- **Outils de collaboration** : Git, GitHub.
